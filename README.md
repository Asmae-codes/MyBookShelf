# MyBookshelf

**My personal library project** – a front-end challenge to create a modern, interactive library website where users can add, delete, and explore books. Built with **HTML5**, **CSS3**, and **vanilla JavaScript**.

---🌸🌸🌸🌸🌸🌸🌸

## **Project Overview**

MyBookshelf is a personal library web application that allows users to:

- Add new books with details such as:
  - Title
  - Author
  - Short summary
  - Publication date
  - Written date
  - Category
  - Optional cover image
  - Optional rating
  - Document link (PDF or URL)

- View all books in a **Library section** with dynamically generated book cards.
- Search books by title or author using a search bar.
- Delete books from the library.
- Display default books with covers, summaries, and links.
- Responsive design for both **desktop** and **mobile** screens.
- Smooth user experience with **modals**, buttons, and interactive elements.

---🌸🌸🌸🌸🌸🌸🌸

## **Features**

1. **Add Book Modal** – a dedicated window to input new book information.
2. **Library Section** – displays all books dynamically.
3. **Search Functionality** – filter books by title or author.
4. **Book Cards** – each card shows:
   - Cover image
   - Title
   - Author
   - Summary
   - Publication and Written dates
   - Category
   - Rating (stars)
   - Clickable document link
5. **Responsive Design** – works on desktop and mobile devices.
6. **Local Storage Integration** – books added by the user remain even after refreshing the page.
7. **Professional Colors & Layout** – orange-to-maroon gradient, clean typography, and modern card design.
8. **Footer Section** – provides information about the website and developer.

---🌸🌸🌸🌸🌸🌸🌸

## **Technologies Used**

- **HTML5** – semantic and accessible markup
- **CSS3** – modern styling with Flexbox, Grid, transitions, and gradient colors
- **JavaScript (Vanilla)** – dynamic rendering, DOM manipulation, event handling, and local storage
